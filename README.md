# Car-Anti-Theft-with-Facial-Recognition
Car anti-theft system designed for Rasperry Pi with in-cabin camera based facial recognition and telegram-based live alerts. The anti-theft system also includes a fail-safe door lock, electronic immobilizer and a sound alarm system.
